# InAmigos NGO Awareness Webpage

## Overview
This project is a simple and structured webpage created to present the work, initiatives, and impact of an NGO in a clear and accessible way.

## Features
- Clean and readable layout
- Sections for About, Projects, Impact, and Gallery
- Real images to reflect actual work
- Smooth navigation and basic interactivity

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
This webpage was built as part of an internship assignment to demonstrate the ability to present information clearly while maintaining a clean and user-friendly design.

## How to Run
1. Download or clone the repository  
2. Open `index.html` in any web browser  

## Note
The content and visuals are organized to reflect realistic NGO activities and focus on clarity rather than complexity.
